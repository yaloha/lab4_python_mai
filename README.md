# Лабораторная работа №5

## Описание

Реализован вариант с библиотекой книг.

Можно:
* добавлять и удалять книги.
* производить поиск по автору, жанру, году и ISBN.
* использовать индексы для быстрого поиска.
* работать с коллекциями книг (`BookCollection`).
* работать с электронными книгами (`EBook`).
* использовать симуляцию случайных событий над библиотекой (`run_simulation`).

---

## Как запустить


- Запуск программы:
   ```
   uv init
   uv run src/main.py
   ```
    или
   ```bash
   python src/main.py
   ```
- Запуск тестов (pytest):
   ```bash
   pytest -v
   ```

## Основные классы

| Класс            | Описание                                                                                |
| ---------------- |-----------------------------------------------------------------------------------------|
| `Book`           | Книга, имеющая автора, название, год, жанр, ISBN                                        |
| `EBook`          | Электронная книга, наследник `Book`, добавлены размер, количество страниц, формат файла |
| `BookCollection` | Коллекция книг с поддержкой индексации и срезов                                         |
| `IndexDict`      | Индексы по ISBN, автору, году и жанру                                                   |
| `Library`        | Библиотека содержащая коллекцию и индексы, поддерживающая поиск и обновленик индексов   |

Где это возможно и логично поля классов инкапсулированы

---

## Симуляция

Функция `run_simulation(steps: int, seed: int)` производит случайные действия над библиотекой:

* добавление книги,
* удаление случайной книги,
* поиск по автору/жанру/году,
* получение книги по несуществующему ISBN,
* обновление индексов.
